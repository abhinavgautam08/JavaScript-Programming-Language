Variables & Data Types

Var, let & const

Var (variable) before 2015 (ES5): can be redeclared and updated. It is a global scope variable.

Example:

var age = 21;
console.log("My age is " + age);

let (variable) after 2015 (ES6): can be updated but cannot be redeclared in the same scope. It is a block scope variable.

Example:

let fullName = "Abhinav Adarsh";
console.log("full name is " + fullName);

const (variable) after 2015 (ES6): cannot be redeclared or updated. It is a block scope variable.

Example:

const age = 21;
console.log("age is " + age);
