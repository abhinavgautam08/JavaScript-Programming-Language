# JavaScript Programming Language

This project is a simple beginner-friendly JavaScript learning page.

## How to view it locally
1. Open the project folder in your browser or run a simple local server.
2. Open index.html in a browser.
3. If you want the files section to load automatically, serve the folder with a local server.
## Features
- Beginner-friendly JavaScript examples
- Page shows the code of JavaScript files
- Files are loaded automatically from the project folder

## Notes
This is a simple learning project and not a full JavaScript framework app.
